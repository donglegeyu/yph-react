# 自动化开发流程

> Last updated: 2026-04-15

## 核心原则

1. **逐步自动化执行** - 小步迭代
2. **明确路径范围** - 确定工作边界
3. **回退机制** - 用「撤回」命令回退不需要的变更
4. **交付物管理** - 同步到原位置和制品目录
5. **历史追踪** - 用记忆文件记录

## 执行步骤

### 1. 需求确认
- 明确任务范围
- 确定交付路径

### 2. 逐步执行
- 每个步骤独立验证
- 及时截图反馈（用户偏好）

### 3. 回退控制
- 用「撤回」命令回退变更
- 确保可逆性

### 4. 交付同步
- 原位置更新
- 制品目录备份

### 5. 记忆更新
- 记录关键决策
- 更新进度状态
