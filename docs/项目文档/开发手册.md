## 2. 技术架构

### 2.1 架构分层

```
┌─────────────────────────────────────────────────────────────┐
│                      客户端层 (Browser)                     │
│         Vue 3 + TypeScript + Ant Design Vue                 │
└────────────────────────────┬────────────────────────────────┘
                             ▼ HTTP/REST
┌─────────────────────────────────────────────────────────────┐
│                      网关层 (Gateway)                        │
│              认证 / 鉴权 / 限流 / 路由分发                   │
└────────────────────────────┬────────────────────────────────┘
                             │
┌─────────────────────────────────────────────────────────────┐
│                      业务服务层 (Service)                    │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │ 材料申请服务 │  │ 采购管理服务 │  │  菜单服务   │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
└────────────────────────────┬────────────────────────────────┘
                             │
┌─────────────────────────────────────────────────────────────┐
│                      数据访问层 (DAO)                        │
│              MyBatis-Plus + Repository Pattern               │
└────────────────────────────┬────────────────────────────────┘
                             │
┌─────────────────────────────────────────────────────────────┐
│                      数据存储层 (DB)                        │
│                     MySQL + Redis Cache                     │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 系统拓扑

```
┌─────────────────────────────────────────────────────────────┐
│                        客户端层                             │
│              Web Browser / Mobile / PC                      │
└────────────────────────────┬────────────────────────────────┘
                             ▼ HTTPS
┌─────────────────────────────────────────────────────────────┐
│                      负载均衡层                             │
│                    Nginx / F5                               │
└────────────────────────────┬────────────────────────────────┘
                             │
     ┌────────────────────────┼────────────────────────┐
     │                        │                        │
     ▼                        ▼                        ▼
┌─────────────┐      ┌─────────────┐           ┌─────────────┐
│  前端集群   │      │  后端集群   │           │   缓存集群  │
│ material-   │      │ material-   │           │   Redis     │
│   system    │      │   server    │           │  Cluster    │
└─────────────┘      └──────┬──────┘           └─────────────┘
                             │
                             ▼
                   ┌─────────────┐
                   │   MySQL     │
                   │  主从架构   │
                   └─────────────┘
```

### 2.3 技术选型理由

| 技术 | 选型 | 理由 |
|------|------|------|
| **Vue 3** | Composition API | 更好的逻辑复用、TypeScript 支持 |
| **Vite** | ESM + HMR | 极速启动、热更新体验好 |
| **Pinia** | Vue 3 官方推荐 | 比 Vuex 更轻量、TypeScript 支持好 |
| **Spring Boot 3** | Java 17 | 虚拟线程、性能提升 |
| **MyBatis-Plus** | ORM 框架 | CRUD 零 SQL、代码生成、逻辑删除 |
| **MySQL 8** | 关系数据库 | 成熟稳定、功能强大 |

### 4.2 核心组件说明

#### 4.2.2 FirstSidebar.vue

一级侧边栏（宽度 126px，背景 `#231F1D`），使用 IconPark sprite 图标。

**状态管理**：

- 使用 `useAppStore` 的 `businessMenus` 渲染业务菜单
- 使用 `customNavMenus` 控制显示的自定义导航

#### 4.2.3 SecondSidebar.vue

二级侧边栏（宽度 200px），配合 `stores/app.ts` 的 `secondMenusMap` 使用：

- 支持 2 层菜单（父级 + children）
- 不支持 3 层嵌套
- 选中色：`#FF5722` / `#FBE9E7`

#### 4.2.4 TabBar.vue

顶部标签栏，使用 app store 的 `tabs` / `activeTabKey`，持久化到 localStorage。

#### 4.2.5 BuildingView.vue

"页面开发中"占位页，无映射路由的兜底组件。

#### 4.2.6 ListPageTemplate.vue ⭐

**统一列表页模板**，所有列表页必须使用此组件，不直接使用 `a-table`。

**功能特性**：

- 筛选区（FilterForm）
- 工具栏（toolbar-actions slot）
- 表格（自动计算 scroll.y）
- 分页
- 列设置（可搜索、checkbox 选择列、持久化到 localStorage）
- 树形表格：表头加号控制全部展开/收起

**详细规范**: 见 [第5章 页面模板](#5-页面模板)

#### 4.2.7 FilterForm.vue ⭐

通用筛选表单组件，支持多种表单项类型。

**响应式断点**（基于 `.main-content` 宽度减去 padding）：

| 内容区宽度 | 列数 | span |
|------------|------|------|
| < 560px | 1列 | `xs:24` |
| 560-860px | 2列 | `sm:12` |
| 860-1200px | 3列 | `md:8` |
| ≥ 1200px | 4列 | `lg:6` |

**收起/展开规则**：

| 状态 | 显示规则 |
|------|----------|
| 收起 | 2行 × 列数 - 1（按钮占1格） |
| 展开 | 显示全部筛选项 + 按钮填满最后一行 |

**详细规范**: 见 [第5章 页面模板](#5-页面模板)

#### 4.2.8 ActionCell.vue

统一渲染表格操作按钮。

**使用方式**：

```vue
<ActionCell :buttons="getActionButtons(record)" />

<script setup>
```

```java
@Service

public class MaterialApplicationServiceImpl 
    extends ServiceImpl<MaterialApplicationMapper, MaterialApplication> 
    implements MaterialApplicationService {
    
    @Override
    public Page<MaterialApplication> listPage(Integer current, Integer size, String materialName) {
        Page<MaterialApplication> page = new Page<>(current, size);
        LambdaQueryWrapper<MaterialApplication> query = new LambdaQueryWrapper<>();
        if (materialName != null && !materialName.isEmpty()) {

        query.orderByDesc(MaterialApplication::getCreateTime);
        return this.page(page, query);
    }
}
```
