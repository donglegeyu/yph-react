# 调试教训与规范

> 隶属：功能开发文档 | 版本：v3.0 | 更新：2026-04-23

---

## 表格相关

| 问题 | 解决方案 |
|------|----------|
| 垂直充满 | `.table-card` 设置 `flex: 1`，用 ResizeObserver 计算 `scroll.y` |
| 表头错位 | `.ant-table-body::-webkit-scrollbar { width: 0 }` |
| show-total 白屏 | 必须用 `:show-total`（函数绑定） |

---

## 图标相关

| 问题 | 解决方案 |
|------|----------|
| IconPark sprite 静态颜色 | `stroke="#333"` 改为 `stroke="currentColor"` |
| `star-fill` 图标 | `fill-opacity` 改为 `1` |

---

## MySQL 相关

| 问题 | 解决方案 |
|------|----------|
| JDBC 乱码 | 参数用 `characterEncoding=utf8`（不是 utf8mb4） |
| 连接池 | 已弃用 Druid，改用 HikariCP |
| Docker 环境 | MySQL(3307) + Backend(8081) |

---

## Vue 响应式相关

| 问题 | 解决方案 |
|------|----------|
| watch 同值重复触发 | 使用 `JSON.stringify` 跳过同值比较 |
| watch 同步刷新重复渲染 | 使用 `flush: 'post'` 选项 |
| computed 单向驱动 | 复杂状态用 computed 替代 watch |

---

## VO/Entity 字段不一致

| 问题 | 解决方案 |
|------|----------|
| 前端字段名与后端不一致 | 统一使用相同的字段名，如 `menuKey` 而非混用 `key` 和 `menuKey` |

## 菜单创建 Checklist（新菜单 → 页面）

### Step 1: 创建 Vue 页面
在 `src/views/common/` 下按菜单名建 `.vue` 文件（参考 `JiuHaoHang.vue` 模板）。

### Step 2: 注册路由
在 `pathComponentMap.ts` 中添加 `'/菜单路径': '@/views/common/菜单.vue'`。
注意：路径前缀斜杠不能重复（如 `/dao-hang` 不是 `//dao-hang`）。

### Step 3: 注册 router/index.ts
在 `DYNAMIC_ROUTES` 中添加（保持同步）。

### Step 4: 修复数据库路径（如需要）
如果菜单路径多了斜杠（如 `//dao-hang` → `/dao-hang`），在数据库中修正。

### Step 5: 验证
刷新页面，点击菜单，导航菜单应该有内容而不是"页面开发中"。
