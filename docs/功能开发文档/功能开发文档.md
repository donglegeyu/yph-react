## 用户约定

- **页面模板**：列表页统一用 **SmartListTemplate**（增强版）或 ListPageTemplate（标准版）
- **Docker 优先**：安装服务端工具优先 Docker
- **看图脚本**：`node ~/.workbuddy/skills/see-image/scripts/see-image.mjs <路径>`
- **网站链接**：生成时同时显示 localhost 和局域网 IP 两个地址
- **Git 推送**：使用 SSH URL（`git@github.com:xxx/xxx.git`），不要用 HTTPS
- **子文档规范**：项目开发相关文档以子文档形式新建在 `docs/` 目录下，命名为 `功能开发文档.XXX.md` 格式
