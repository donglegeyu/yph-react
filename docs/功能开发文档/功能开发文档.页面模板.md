## 双模板体系

### 模板选择指南

| 模板 | 适用场景 | 核心特性 |
|------|----------|----------|
| **SmartListTemplate** | **推荐使用** | 字段级视图管理、ColumnSettingsPanel、三端同步 |

---

## SmartListTemplate.vue ✅

**增强版列表页模板**（1206行，系统最大组件）。

### 核心特性

- 字段级视图管理（新增/编辑/删除视图，拖拽排序）
- ColumnSettingsPanel 列设置面板集成
- **三端同步**：defaultColumnFields + filterItems + dialogFilterOptions 自动同步
- computed 单向驱动字段同步（2026-04-23 优化）

### 三端同步规则

当列表增加字段时，必须同步修改三处：
1. `defaultColumnFields` - 表格列定义
2. `filterItems` - 筛选表单字段
3. `dialogFilterOptions` - 视图管理选项

### 标准模板

- `src/views/list/MaterialList.vue`（SmartListTemplate 示例）
- `src/views/common/ConstructionApplicationList.vue`（SmartListTemplate + ColumnSettingsPanel）

## ListPageTemplate.vue ✅

**统一列表页模板**，所有列表页必须使用此组件。

### 功能特性

- 页面标题区（含 title-actions slot）
- 筛选区（FilterForm）
- 工具栏（toolbar-actions slot）
- 表格（自动计算 scroll.y，响应 resize）
- 分页（支持 total/page-size-changer/quick-jumper）
- 列设置（可搜索、checkbox 选择列、持久化到 localStorage）
- 行选择（rowSelection）
- 树形表格：表头加号控制全部展开/收起

### Props 定义

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| title | string | '' | 页面标题 |
| filterItems | any[] | [] | 筛选项配置 |
| filter-model-value | object | {} | 筛选表单绑定值（v-model） |
| columns | any[] | [] | 表格列配置 |
| dataSource | any[] | [] | 表格数据源 |
| loading | boolean | false | 加载状态 |
| pagination | boolean/object | true | 分页配置，false 禁用 |
| rowSelection | object | null | 行选择配置 |
| rowKey | string/function | 'id' | 行唯一标识 |

### Emits 事件

| 事件 | 参数 | 说明 |
|------|------|------|
| search | searchData | 搜索触发 |
| reset | - | 重置触发 |
| add | - | 新增触发 |
| refresh | - | 刷新触发 |
| change | pagination | 表格变化/分页变化 |
| update:pagination | { current, pageSize } | 分页更新 |

### Slots

| Slot | 说明 |
|------|------|
| title-actions | 页面标题区右侧操作按钮 |
| toolbar-actions | 工具栏按钮区 |
| bodyCell | 自定义单元格（透传 a-table） |
| headerCell | 自定义表头（透传 a-table） |

### 使用方式

```vue
<ListPageTemplate
  v-model:filter-model-value="filterParams"
  :filter-items="filterItems"
  :columns="columns"
  :data-source="dataSource"
  :loading="loading"
  :title="pageTitle"
  :pagination="paginationConfig"
  :row-selection="rowSelection"
  @search="fetchData"
  @reset="handleReset"
  @change="handleTableChange"
>
  <template #title-actions>
    <a-button>标题区按钮</a-button>
  </template>
  <template #toolbar-actions>
    <a-button type="primary">新增</a-button>
  </template>
  <template #bodyCell="{ column, record }">
    <!-- 自定义单元格 -->
  </template>
</ListPageTemplate>
```

### 新建页面 Checklist

#### Step 1: 创建 Vue 页面
在 `src/views/common/` 下按菜单名建 `.vue` 文件（参考 `JiuHaoHang.vue` 模板）。

#### Step 2: 注册路由
在 `pathComponentMap.ts` 中添加 `'/菜单路径': '@/views/common/菜单.vue'`。
注意：路径前缀斜杠不能重复（如 `/dao-hang` 不是 `//dao-hang`）。

#### Step 3: 修复数据库路径（如需要）
如果菜单路径多了斜杠（如 `//dao-hang` → `/dao-hang`），在数据库中修正。

#### Step 4: 验证
刷新页面，点击菜单，导航菜单应该有内容而不是"页面开发中"。

## FilterForm.vue 筛选表单规范

### 布局规则

**响应式断点**（基于 `.main-content` 宽度减去 padding）：

| 内容区宽度 | 列数 | span |
|------------|------|------|
| < 560px | 1列 | `xs:24` |
| 560-860px | 2列 | `sm:12` |
| 860-1200px | 3列 | `md:8` |
| ≥ 1200px | 4列 | `lg:6` |

**收起/展开规则**：

| 状态 | 显示规则 |
|------|----------|
| 收起 | 2行 × 列数 - 1（按钮占1格） |
| 展开 | 显示全部筛选项 + 按钮填满最后一行 |

### 组件结构

```
.filter-form (bg:#fff, padding:0.625rem 0.75rem, radius:0.38rem)
└── a-row :gutter="24"
    ├── a-col (v-for 筛选项) :span响应式
    ├── a-col (占位) :span响应式
    └── a-col (按钮) :span响应式
```

### 样式代码

```css
/* 容器 */
.filter-form {
  background: #fff;
  padding: 0.625rem 0.75rem;
  border-radius: 0.38rem;
}

/* 表单项 */
.filter-item {
  display: flex;
  flex-direction: column;
  width: 100%;
}

/* 标签 */
.filter-item .ant-form-item-label {
  width: 80px;
  flex-shrink: 0;
}

.filter-item .ant-form-item-label > label {
  justify-content: flex-end;
  margin: 6px 0;
}

/* 输入控件 */
.filter-item .ant-form-item-control {
  flex: 1;
  min-width: 0;
}

.filter-item .ant-input,
.filter-item .ant-select,
.filter-item .ant-picker {
  width: 100%;
  margin: 6px 0;
}

/* 选中项颜色 */
.filter-item .ant-select-selection-item {
  color: #F95914;
}

/* 按钮组 */
.btn-col .ant-space {
  display: flex;
  justify-content: flex-end;
}

.btn-col .ant-space-item {
  margin: 6px 0;
}

.btn-col .ant-btn {
  box-shadow: none;
}
```

### 关键样式参数

| 属性 | 值 | 说明 |
|------|-----|------|
| 容器背景 | `#FFFFFF` | 白色背景 |
| 容器内边距 | `10px 12px` | 上下 0.625rem，左右 0.75rem |
| 容器圆角 | `0.38rem` | 顶部圆角 |
| 列间距 | `24px` | a-row gutter |
| 标签宽度 | `80px` | 固定宽度右对齐 |
| 标签/输入框垂直间距 | `6px` | 上下居中对齐 |
| 选中项颜色 | `#F95914` | 主色 |
| 按钮阴影 | `none` | 去掉默认阴影 |
