## 目录结构

```
src/components/
├── layout/                    # 布局组件
│   ├── MainLayout.vue         # 主布局
│   ├── FirstSidebar.vue       # 一级侧边栏（126px）
│   ├── SecondSidebar.vue      # 二级侧边栏（200px）
│   ├── TabBar.vue            # 顶部标签栏
│   ├── MoreMenuDrawer.vue    # 更多菜单抽屉
│   ├── CustomNavPanel.vue    # 自定义导航面板
│   └── BuildingView.vue      # 占位页"页面开发中"
├── ListPageTemplate.vue      # 列表页模板（标准版）
├── SmartListTemplate.vue     # 智能列表模板（增强版，1206行）
├── FilterForm.vue            # 筛选表单
├── IconSelect.vue           # 图标选择器
├── ColumnSettingsPanel.vue   # 列设置面板（316行）
├── ActionCell.vue           # 操作列按钮组
└── FilterOptionsDrawer.vue  # 视图筛选条件抽屉（290行）
```

## 布局组件

### MainLayout.vue
主布局组件，组合所有布局组件。

### FirstSidebar.vue
一级侧边栏（宽度 126px，背景 `#231F1D`），使用 IconPark sprite 图标。

### SecondSidebar.vue
二级侧边栏（宽度 200px），配合 `stores/app.ts` 的 `secondMenusMap` 使用：
- 支持 2 层菜单（父级 + children）
- 不支持 3 层嵌套
- 选中色：`#FF5722` / `#FBE9E7`
- 支持固定/悬浮两种模式

### TabBar.vue
顶部标签栏，使用 app store 的 `tabs` / `activeTabKey`，持久化到 localStorage。

### BuildingView.vue
"页面开发中"占位页，无映射路由的兜底组件。

## 通用组件

### SmartListTemplate.vue（1206行）
**增强版列表页模板**，详见 `功能开发文档.页面模板.md`
- 字段级视图管理
- ColumnSettingsPanel 集成
- 三端同步支持

### ListPageTemplate.vue
**标准列表页模板**，详见 `功能开发文档.页面模板.md`

### ColumnSettingsPanel.vue（316行）
**列设置面板组件**
- 功能：可搜索的列选择器，支持 checkbox 多选列
- 拖拽排序字段顺序
- 显示/隐藏字段勾选
- 用途：SmartListTemplate 的列设置功能
- 示例：`MaterialList.vue`、`ConstructionApplicationList.vue`

### FilterOptionsDrawer.vue（290行）
**视图筛选条件抽屉组件**
- 功能：视图管理的筛选条件配置
- 支持视图新增 / 编辑两种模式
- 拖拽排序筛选条件
- 与 SmartListTemplate 配合使用

### FilterForm.vue
通用筛选表单组件，支持多种表单项类型（input/select/date/daterange）。

### IconSelect.vue
图标选择器组件，支持从 IconPark 选择图标并预览。

### ActionCell.vue
操作列按钮组，统一渲染表格操作按钮。
