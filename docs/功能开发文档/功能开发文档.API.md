## 施工申请 API 详情

### GET /api/construction-applications
查询施工申请列表

**参数**：
| 参数 | 类型 | 说明 |
|------|------|------|
| current | int | 当前页码 |
| size | int | 每页条数 |
| applicationNo | string | 申请单号（模糊查询） |
| constructionName | string | 施工项名称（模糊查询） |
| applicant | string | 申请人（模糊查询） |
| status | string | 状态 |

**响应**：
```json
{
  "code": 200,
  "data": {
    "records": [...],
    "total": 100,
    "current": 1,
    "size": 20
  }
}
```

### POST /api/construction-applications
创建施工申请

### PUT /api/construction-applications/{id}
更新施工申请

### DELETE /api/construction-applications/{id}
删除施工申请

## 材料视图 API 详情

### GET /api/material-views
查询材料视图列表

**参数**：
| 参数 | 类型 | 说明 |
|------|------|------|
| userId | long | 用户ID |
| pageType | string | 页面类型（如 `material`） |

### POST /api/material-views
创建视图

### PUT /api/material-views/{id}
更新视图

### DELETE /api/material-views/{id}
删除视图
