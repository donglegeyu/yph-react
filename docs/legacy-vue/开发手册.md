### GET /api/materials

**获取材料列表**

**请求参数:**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| keyword | string | 否 | 搜索关键词 |
| pageNum | integer | 否 | 页码 |
| pageSize | integer | 否 | 每页数量 |

**响应示例:**

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
