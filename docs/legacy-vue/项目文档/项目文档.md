## 1. 项目概述

### 1.1 项目简介

材料申请管理系统是一个面向集团企业的物资采购管理平台，支持多租户架构，提供完整的材料申请、审批、采购订单管理等功能。系统采用前后端分离架构，前端基于 Vue 3 + TypeScript，后端基于 Spring Boot 3 + MyBatis-Plus。

### 1.2 核心功能

| 模块 | 功能描述 |
|------|----------|
| **材料申请** | 材料申请单的创建、编辑、提交、审批流程管理 |
| **采购管理** | 采购需求单、采购订单的创建和管理 |
| **供应商管理** | 供应商档案、资质管理、评分体系 |
| **施工项管理** | 施工项字典、价格策略配置 |
| **审批流程** | 可配置的动态审批流程，支持多级审批 |
| **报表统计** | 采购统计、供应商分析等报表功能 |

### 1.3 技术指标

| 指标 | 目标值 |
|------|--------|
| **响应时间** | P95 < 500ms |
| **QPS** | 1000-5000 (可扩展至 10000+) |
| **并发用户** | 500+ |
| **数据规模** | 支持百万级数据 |
| **可用性** | 99.9% |

**SmartListTemplate Props 接口**

```typescript
interface Props {
  title?: string                  // 页面标题
  fields?: FieldDefinition[]      // 字段定义（同时作为表格列和筛选项）
  loading?: boolean               // 加载状态
  dataSource?: any[]              // 表格数据
  viewEndpoint?: string           // 视图接口端点（默认 /api/views）
  viewType?: string               // 视图类型标识（默认 default）
  rowKey?: string | Function      // 行唯一标识
}
```

**使用示例（ListPageTemplate）**

```vue
<template>

    :filter-items="filterItems"
    :columns="columns"

    @reset="handleReset"
    @add="handleAdd"
  >
    <template #toolbar-actions>
      <a-button type="primary" @click="handleAdd">新增</a-button>
    </template>
    <template #bodyCell="{ column, record }">
      <template v-if="column.key === 'action'">

</template>
```

import { defineStore } from 'pinia'

export const useExampleStore = defineStore('example', () => {
  // 状态
  const data = ref<any[]>([])
  const loading = ref(false)
  
  // 计算属性
  const dataCount = computed(() => data.value.length)
  
  // 方法
  async function fetchData() {
    loading.value = true
    try {
      const res = await fetch('/api/example')
      const json = await res.json()
      if (json.code === 200) {
        data.value = json.data || []
      }
    } finally {
      loading.value = false
    }
  }
  
  return {
    data,
    loading,
    dataCount,
    fetchData
  }
})
