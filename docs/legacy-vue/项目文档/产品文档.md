## 1. 产品简介

### 1.1 产品概述

物资管理系统是一款面向企业级用户的采购与物资管理平台，支持材料申请、采购订单、供应商管理、财务结算等核心业务流程。系统采用前后端分离架构，支持多租户数据隔离，可灵活扩展至千万级数据量。

### 1.2 主要特性

| 特性 | 说明 |
|------|------|
| **多端适配** | 支持 Web / PC / Mobile 多端访问 |
| **多租户支持** | 支持标准、高级、私有化三种部署模式 |
| **实时审批** | 基于 WebSocket 的审批流程实时通知 |
| **动态工作流** | 可配置的多级审批流程 |
| **数据隔离** | 基于 tenant_id 的数据安全保障 |
| **可扩展架构** | 支持从单体平滑演进至微服务架构 |

### 1.3 技术规格

| 项目 | 技术方案 |
|------|----------|
| **前端框架** | Vue 3 + Vite + TypeScript |
| **UI 组件库** | Ant Design Vue 4.x |
| **状态管理** | Pinia 2.x |
| **后端框架** | Spring Boot 3.2.5 |
| **ORM 框架** | MyBatis-Plus 3.5.6 |
| **数据库** | MySQL 8 |
| **连接池** | HikariCP |
| **推荐分辨率** | 1920×1080 |
