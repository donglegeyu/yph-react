## 1. 组件结构

```
FirstSidebar（容器）
├── .brand-area（品牌区）
├── .system-menu-top（系统菜单-上：首页、收藏）
│   └── .divider（分割线：菜单 > 1 时显示）
├── .business-menus（业务菜单：调接口，按模块区分显示）
└── .system-menu-bottom（系统菜单-下）
    └── .nav-divider（分割线：存在系统菜单时显示）
    └── 系统菜单项...
    └── .person-info（个人信息）
```

## 1.1 分割线显示规则

| 位置 | 条件 | 显示 |
|------|------|------|
| 系统菜单-上下方 (.divider) | `firstMenus.length > 1` | ✅ 显示 |
| 系统菜单-上下方 (.divider) | `firstMenus.length ≤ 1` | ❌ 隐藏 |
| 系统菜单-下上方 (.nav-divider) | `systemBottomMenus.length > 0` | ✅ 显示 |
| 系统菜单-下上方 (.nav-divider) | `systemBottomMenus.length === 0` | ❌ 隐藏 |

## 9. 二级菜单面板交互模式

### 9.1 悬浮模式 vs 固定模式

| 模式 | 触发条件 | 面板行为 |
|------|----------|----------|
| 悬浮模式 | 默认状态 | 鼠标离开后延迟 100ms 隐藏 |
| 固定模式 | 点击底部"固定菜单"按钮 | 始终显示，切换按钮可取消 |

### 9.2 交互规则

**显示逻辑**：
```
显示二级面板 = 鼠标悬停一级菜单 || 面板固定状态
```

**重要原则**：
- Tab 切换页面时，**不自动显示**二级面板
- 只有鼠标**实际悬停**在一级菜单时才显示二级面板
- 二级菜单的选中高亮状态始终保持正确
- **用户展开的菜单不自动收起**，保留展开状态直到用户手动收起
- **悬停一级菜单时**，自动展开第一个有子菜单的二级菜单

**展开状态保持**：
| 操作 | expandedKeys 行为 |
|------|-------------------|
| 展开菜单 A | 添加 A 到列表 |
| 展开菜单 B | 添加 B 到列表（A 保持） |
| 点击二级/三级菜单 | 保留已展开的菜单 |
| Tab 切换页面 | 保留已展开的菜单 |
| 切换一级菜单 | 保留已展开的菜单 |
| 点击首页 | 清空展开状态 |

### 9.3 状态管理

| 状态变量 | 文件位置 | 说明 |
|----------|----------|------|
| `activeFirstMenu` | app.ts | 当前选中的一级菜单 |
| `isFirstMenuHovering` | app.ts | 鼠标是否悬停在一级菜单 |
| `secondSidebarHovered` | app.ts | 二级面板悬浮状态 |
| `secondSidebarFixed` | app.ts | 二级面板固定状态 |

### 9.4 交互流程图

```
┌─────────────────────────────────────────────────────────────┐
│                     用户操作                                 │
└─────────────────────────────────────────────────────────────┘
                              │
        ┌─────────────────────┼─────────────────────┐
        ▼                     ▼                     ▼
   ┌─────────┐          ┌─────────┐          ┌─────────┐
   │ 点击一级 │          │ 悬停一级 │          │ Tab切换 │
   │  菜单   │          │  菜单   │          │  页面   │
   └────┬────┘          └────┬────┘          └────┬────┘
        │                    │                    │
        ▼                    ▼                    ▼
   设置 activeFirstMenu  设置 isFirstMenuHovered  调用 navigateToPath
        │                    │                    │
        ▼                    ▼                    ▼
   显示二级面板        检查 isFirstMenuHovered    同步 activeKey/expandedKeys
                                                  │
                                                  ▼
                                            不显示二级面板
```

### 9.5 关键代码位置

| 功能 | 文件 | 函数/变量 |
|------|------|----------|
| 鼠标进入业务菜单 | FirstSidebar.vue | `handleBusinessMenuHover()` |
| 鼠标离开业务菜单 | FirstSidebar.vue | `handleBusinessMenuLeave()` |
| 一级菜单切换监听 | SecondSidebar.vue | `watch(activeFirstMenu)` |
| 延迟隐藏面板 | app.ts | `delayHideSidebar()` |
| 取消隐藏面板 | app.ts | `cancelHideSidebar()` |

### 9.6 收藏功能

- 收藏图标显示在每个二级/三级菜单右侧
- 悬停时显示，点击切换收藏状态
- 已收藏菜单显示 `#F95914` 实心星形图标
- 收藏面板支持拖拽排序
