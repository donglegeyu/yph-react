# 主题设置

> 本文档记录主题设置页面的功能开发规范和使用说明。

---

## 1. 功能概述

主题设置页面（ComponentPreview.vue）是系统的 Design Token 可视化管理平台，提供以下核心功能：

- **全局样式管理**：编辑和预览 Design Token（颜色、间距、圆角等）
- **业务组件预览**：展示自定义业务组件的样式效果
- **基础组件预览**：展示 Ant Design 基础组件的样式效果

### 2.1 全局样式 Tab

```
┌──────────┬──────────────────────────┬───────┬──────────────────┐
│  分类导航  │       Token 列表        │ 分隔符 │    实时预览      │
│  (200px)  │        (600px)          │  (1px) │     (flex)       │

```

**分类导航**：左侧菜单，展示 Token 分类（基础色、文本色、背景色等）

**Token 列表**：
- ColorScale 组件：展示色阶（primary-1 到 primary-10）
- Token 列表项：显示颜色预览、名称、key、输入框

**实时预览**：
- 选择色阶时：显示颜色详情面板 + 无障碍检查
- 未选择时：显示按钮、标签、文本等基础预览

### 2.2 业务组件 Tab

```
┌──────────┬──────────────────────────┬──────────┐
│  组件导航  │       组件演示区         │ Token 列表│

```

**组件导航**：ActionCell、IconSelect、GlobalSelect、FilterForm

**组件演示区**：每个组件的交互式演示

**Token 列表**：右侧显示当前组件使用的 Design Token

## 4. 数据结构

### 4.1 DesignToken

```typescript
interface DesignToken {
  id: number              // Token ID
  categoryId: number      // 分类 ID
  categoryCode: string    // 分类编码
  name: string           // Token 名称（如 "主色"）
  tokenKey: string        // Token 变量名（如 "--primary-color"）
  tokenType: string      // Token 类型（color/number/spacing/text）
  defaultValue: string   // 默认值
  currentValue: string    // 当前值
  description?: string   // 描述/分组名
  isAntDesignToken: boolean   // 是否为 Ant Design Token
  antDesignTokenName?: string // Ant Design Token 名称
  sortOrder: number      // 排序
  updatedAt?: string     // 更新时间
}
```

### 4.2 TokenCategory

```typescript
interface TokenCategory {
  id: number
  name: string           // 分类名称
  code: string           // 分类编码
  sortOrder: number      // 排序
  tokenCount?: number    // Token 数量
}
```

## 7. 状态管理

### 7.1 useDesignTokens 状态

```typescript
// 响应式状态
tokens: Ref<DesignToken[]>        // Token 列表
categories: Ref<TokenCategory[]> // 分类列表
loading: Ref<boolean>            // 加载状态
hasLoaded: Ref<boolean>          // 是否已加载

// 方法
loadTokens()                      // 加载 Token
updateToken(id, value)            // 更新单个 Token
saveAndApply()                    // 保存并应用
resetToDefault()                  // 重置为默认
```

### 7.2 页面状态

```typescript
// Tab 状态
activeTab: ref('tokens' | 'components' | 'antd-components')

// 全局样式 Tab

// 业务组件 Tab
```

## 9. 色阶规范

### 9.1 色阶格式

Token Key 格式：`--{color}-{level}`

示例：
- `--primary-1` 到 `--primary-10`
- `--gray-1` 到 `--gray-10`

### 9.2 色阶含义

| 级别 | 用途 |
|------|------|
| 1 | 最浅色 |
| 2 | 浅色 |
| 3 | 次浅色 |
| 4 | 浅主色 |
| **5** | **主色（基准色）** |
| 6 | 深主色 |
| 7 | 深色 |
| 8 | 次深色 |
| 9 | 深色 |
| 10 | 最深色 |
